Python 基础学习笔记

从变量、流程控制、函数、类到 NumPy 的一组基础练习记录，并附上所有 notebook 代码单元。

March 13, 2026

这部分是我后续做 RAG、Agent 和 API 脚本之前的基础阶段。虽然内容偏基础，但其实后面几乎所有 AI 工程代码都还是建立在这些基本语法和组织方式之上。

1. 我主要覆盖了什么

`CodingBasis/` 下面的几个 notebook，分别对应了 Python 的变量与基础语法、函数、类，以及 NumPy 入门。它们组合起来，基本构成了后续写脚本最常用的那一层能力。

2. 我具体在练什么

  • 变量类型、字符串、数值、布尔值、集合、字典、列表等基本数据结构。
  • 条件判断、循环、类型转换等最常见的控制逻辑。
  • 函数定义、参数传递、返回值设计，以及把逻辑封装成可复用片段。
  • 类、初始化、继承与组合，让代码不只是“能跑”，还更容易组织。
  • NumPy 数组的基本用法，为后续更偏数据和模型的学习打基础。

3. 我当前的理解

我越来越感受到，基础语法本身不难，但真正重要的是“组织方式”。当函数、类和数据结构用得更顺手时，后面做任何模型应用都会更自然，不会因为脚本本身混乱而影响思路。
